# Neon Ridge Arena

A small browser multiplayer shooting game with a large top-down map, server-side movement, shooting, and drivable cars.

## Run

```bash
npm start
```

Open `http://localhost:3000` in two browser tabs to test multiplayer locally.

## Controls

- `WASD` or arrow keys: move / steer
- `Shift`: sprint / boost
- `Mouse`: aim
- `Left click`: shoot
- `F`: enter a nearby car
- `E`: exit a car
- `Space`: brake

For multiplayer across devices, keep the server running and open `http://YOUR_LOCAL_IP:3000` from another device on the same network.
